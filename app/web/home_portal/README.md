# Perk Nation Website (Static)

This site implements the exact page structure from the provided 'WebSite Structure Perk Nation' document:
- Header tabs: Members, Merchants, How it Works, Contact Us, FAQ
- Footer sections: Product / Company / Support
- Cookie banner with Reject/Accept and a link to Disclaimer

## Theme
This is an original Perk Nation theme with an inKind-inspired palette and the publicly-used 'Rethink Sans' font.

## Run locally
Open `index.html` directly, or run:
python3 -m http.server 8091 --directory .

## Forms
Forms store submissions to localStorage for demo. Wire to a backend for production.
